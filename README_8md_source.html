3 ## Overview
4 The aim of the project is to design and develop a modular smart greenhouse that is capable of detecting and responding to a number of changes in environmental conditions in real time. Sensors are used to measure soil moisture, humidity, temperature, UV light and air quality. The system will also incorporate LED lighting, heating and watering systems with automated and manual control. Moreover, the real-time measurements will be displayed on an online web portal enabling the user to view the real-time and historic growth of the plant.
5 
6 The **SmartGreenhouse** will enable users to monitor and manage their plant growth environment remotely, making this an ideal system for travelling professionals and students or those who live in unpredictable climates.

56 ## Build & Install
57 1. Check you have the necessary hardware components and have installed the necessary packages.
58 1. Assemble the greenhouse structure.
59 1. Plant the seeds/bulbs/sprouts in the planter.
60 1. Position the sensors and response systems and feed the connection cables out of the greenhouse.
61 1. Connect the components to the RaspberryPi using the Raspberry Pi header.
62 1. Download the binary software files from GitHub.
63 1. Search for and connect the Raspberry Pi and your access device to the hidden wifi network `rank510iot`, password: `raspberry`.
64 1. Access the SmartGreenhouse webpage.
65 1. Run the system.
66 
67 A more detailed walkthrough can be found [here.](https://github.com/Team12-UofG/SmartGreenhouse/wiki/Build-and-Install-Guides/ "here.")
